💡 ¿Por qué intervenir con IA?
🏫 Las brechas educativas en Colombia tienen una dimensión territorial profunda: los municipios rurales de Cundinamarca concentran los mayores índices de deserción, reprobación y baja cobertura. El sistema educativo oficial carece de herramientas predictivas para anticipar estos riesgos. La IA combinada con análisis geoespacial puede transformar datos dispersos en decisiones pedagógicas precisas.

🧠 Propuesta doctoral
Qué se va a investigar y construir

"Modelo predictivo basado en IA y análisis geoespacial para reducir brechas educativas y digitales en municipios rurales de Cundinamarca, con canales de monitoreo territorial en tiempo real"

— Título oficial de la investigación doctoral
🎯 Componentes del proyecto
🗺️
Modelo geoespacial predictivo
Integra datos de cobertura, distancia, infraestructura y conectividad con indicadores de eficiencia escolar para predecir riesgos de fracaso educativo por municipio.
⚡
Canales de monitoreo en tiempo real
Alertas tempranas para desastres naturales (deslizamientos, inundaciones), rendimiento escolar y riesgo territorial en los 116 municipios de Cundinamarca.
🔓
Dataset abierto de Cundinamarca
Primer dataset educativo geoespacial abierto del departamento — disponible para toda Colombia como insumo para modelos de IA nacionales.
🎛️
Plataforma web para directivos
Dashboard funcional con React + D3.js para secretarías de educación. Protocolo replicable para otros departamentos del país.
🌱 Misiones Minciencias
✅ Misión 1 — Bioeconomía y territorio
IA territorial para gestión educativa en regiones rurales de Cundinamarca — municipios PDET prioritarios.
✅ Misión 2 — Ciencia para la paz
Reducción de brechas educativas como factor de equidad territorial y cohesión social.
📌 ODS 4 + Agenda Digital Colombia 2026
Plan Decenal de Educación + Plan Nacional de Conectividad con métricas medibles.
🛠️ Stack tecnológico
🗺️ GIS + Python — GeoPandas, QGIS, análisis territorial
🤖 Machine Learning — XGBoost, Random Forest, LSTM
📡 APIs oficiales — SINEB, SIMAT, IGAC, DANE
🎛️ Dashboard — React + D3.js + Vercel
🔓 Dataset abierto — GitHub · Zenodo · CKAN
